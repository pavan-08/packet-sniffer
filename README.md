# Packet Sniffer
A Packet Sniffer app made using Python 

# Setup
Change directory into the project after cloning and run,
```
sh setup.sh
```
It will ask for root access

# Run the project
Either do,
```
sudo python GUISniffer.py
```
or
```
sudo ./Packet\ Sniffer
```
